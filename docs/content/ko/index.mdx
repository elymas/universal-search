# Universal Search

**Universal Search**는 팀 규모의 소스 기반 연구 메타 에이전트입니다. 웹, 소셜, 학술, 기술, 한국어 로케일 검색을 단일 쿼리 플레인으로 통합하고, 결과를 하이브리드 LLM 합성을 통해 제공합니다.

## 주요 기능

| 기능 | 설명 |
|---|---|
| **멀티 소스 검색** | 웹, 소셜, 학술, 기술, 한국어 로케일 |
| **하이브리드 합성** | 기본 요약+인용; `/deep`은 멀티 에이전트 심층 연구 |
| **CLI + MCP + 웹 UI** | 다양한 워크플로를 위한 세 가지 인터페이스 |
| **한국어 지원** | 한국어 로케일 검색 및 이중 언어 문서 (EN/KO) |
| **자체 호스팅** | 팀을 위한 Docker Compose 배포 |

## 5분 안에 시작하기

```bash
git clone <repo-url> && cd universal-search
cp .env.example .env        # MEILI_MASTER_KEY, POSTGRES_PASSWORD 등 입력
make compose-up             # Qdrant, Meilisearch, PG, SearXNG, LiteLLM 시작
make build                  # usearch 바이너리 빌드
./cmd/usearch/usearch query "universal search란 무엇인가?"
```

전체 설치 가이드는 [설치](/ko/getting-started/installation)를 참조하세요.
