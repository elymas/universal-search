# 보안 런북

<Callout type="warning">
  **포워드 레퍼런스 — SPEC-SEC-001 미완료**

  전체 보안 런북(인시던트 대응 절차, 에스컬레이션 경로, 사후 분석 템플릿)은 **SPEC-SEC-001** (PR #42, main 브랜치 머지 대기 중)에서 제공됩니다.

  이 페이지는 플레이스홀더입니다. SPEC-SEC-001이 머지되면 확장됩니다.
</Callout>

## 런북에 포함될 내용

SPEC-SEC-001이 머지되면 이 런북에 다음이 포함됩니다:

- **인시던트 탐지** — Universal Search에서 보안 인시던트를 식별하는 방법
- **심각도 분류** — 보안 인시던트에 대한 P0/P1/P2/P3 기준
- **에스컬레이션 경로** — 알림 대상자 및 순서
- **격리 절차** — 피해 범위를 제한하기 위한 즉각적인 조치
- **증거 수집** — 로그 보존 및 포렌식 아티팩트 보존
- **복구 절차** — 보안 인시던트 후 서비스 복원
- **사후 분석 템플릿** — 교훈을 위한 구조화된 검토 형식
