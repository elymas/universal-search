# OWASP ASVS 체크리스트

<Callout type="warning">
  **포워드 레퍼런스 — SPEC-SEC-001 미완료**

  전체 OWASP ASVS(애플리케이션 보안 검증 표준) 체크리스트 및 증거는 **SPEC-SEC-001** (PR #42, main 브랜치 머지 대기 중)에서 제공됩니다.

  이 페이지는 플레이스홀더입니다. SPEC-SEC-001이 머지되면 확장됩니다.
</Callout>

## 체크리스트에 포함될 내용

SPEC-SEC-001이 머지되면 이 페이지에 다음이 포함됩니다:

- Universal Search에 적용 가능한 **ASVS 레벨 2 컨트롤** (웹 API + CLI + MCP)
- 각 컨트롤에 대한 **검증 증거** (테스트 결과, 감사 로그)
- 실패한 컨트롤에 대한 **개선 상태**
- **타사 의존성 감사** 결과

## 현재 상태

SPEC-SEC-001은 현재 Universal Search의 전체 ASVS 레벨 2 감사를 진행 중입니다. SPEC이 머지되면 결과가 이 페이지에 게시됩니다.
