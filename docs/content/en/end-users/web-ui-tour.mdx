## Accessing the Web UI

After starting the stack (`make compose-up`), open:

```
http://localhost:3000
```

## Features

### Search Interface
- Query input with autocomplete
- Locale selector (EN / KO)
- Source filter (web, academic, social, Korean)
- Deep research toggle

### Results View
- Synthesized answer with inline citations
- Citation panel (expand each source)
- Confidence score indicator
- Export to Markdown / JSON

### Query History
- Shared team history timeline
- Full-text search across past queries
- Re-run any past query

### Admin Panel (Operator)
- User management
- Token management
- Adapter configuration
- System health dashboard
