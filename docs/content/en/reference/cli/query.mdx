```
Query the universal search system.

Routes the prompt through intent classification, fan-out to adapters,
and LLM synthesis. Output goes to stdout; progress/errors to stderr.

Usage:
  usearch query [flags] <prompt>

Examples:
  usearch query "latest AI research"
  usearch query --format json --source reddit "golang generics"
  usearch query --timeout 10s "quick lookup"

Flags:
      --format string    output format: text (default), json, or markdown (default "text")
  -h, --help             help for query
      --no-llm           skip LLM client initialisation
      --no-obs           disable observability init (test flag)
      --source string    comma-separated adapter names (empty = all enabled)
      --timeout string   total pipeline deadline (max 5m) (default "30s")

Global Flags:
      --repl   force interactive REPL mode
```
