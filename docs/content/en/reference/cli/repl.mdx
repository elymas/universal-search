# usearch repl

Launch an interactive REPL (Read-Eval-Print Loop) for conversational search sessions.

## Synopsis

```
usearch repl [flags]
```

## Flags

| Flag | Default | Description |
|---|---|---|
| `--locale <code>` | `en` | Default locale for REPL session |
| `--no-history` | false | Disable session history persistence |

## Examples

```bash
# Start REPL
usearch repl

# REPL with Korean locale
usearch repl --locale ko
```

## REPL Commands

Inside the REPL:

| Command | Description |
|---|---|
| `<query text>` | Run a search query |
| `/deep <query>` | Run deep research mode |
| `/history` | Show session history |
| `/sources` | List active adapters |
| `/locale <code>` | Change locale for this session |
| `/help` | Show REPL help |
| `/quit` or `Ctrl+D` | Exit REPL |

## Session Example

```
usearch> what is Raft consensus?
Searching... [web, academic]

Raft is a consensus algorithm designed to be more understandable than Paxos...

Sources: [1] raft.github.io, [2] etcd.io

usearch> /deep compare it to Paxos
Running deep research (this may take 30-120s)...

# Research Report: Raft vs Paxos
...

usearch> /history
[1] what is Raft consensus?       (2026-05-30 14:22)
[2] compare it to Paxos           (2026-05-30 14:25)

usearch> /quit
```

## Notes

- REPL maintains conversational context within the session (subsequent queries can reference prior results)
- Session history is persisted to the shared Meilisearch index unless `--no-history` is used
- Press `Ctrl+C` to cancel a running query without exiting
