# usearch sources

Manage the search adapter registry — list, enable, disable, and test source adapters.

## Synopsis

```
usearch sources <subcommand> [flags]
```

## Subcommands

| Subcommand | Description |
|---|---|
| `list` | List all registered adapters and their status |
| `enable <name>` | Enable an adapter |
| `disable <name>` | Disable an adapter |
| `test <name>` | Test an adapter with a sample query |
| `info <name>` | Show detailed adapter information |

## Examples

```bash
# List all adapters
usearch sources list

# Enable Naver adapter (for Korean-locale)
usearch sources enable naver

# Disable an adapter
usearch sources disable youtube

# Test an adapter
usearch sources test arxiv --query "raft consensus"

# Get adapter details
usearch sources info naver
```
