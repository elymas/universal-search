# CLI Reference

`usearch` is the Universal Search command-line interface. It provides 7 top-level subcommands for search, configuration, history, deep research, source management, authentication, and interactive REPL mode.

## Subcommands

| Subcommand | Description |
|---|---|
| [`query`](/en/reference/cli/query) | Run a search query |
| [`config`](/en/reference/cli/config) | Manage configuration |
| [`history`](/en/reference/cli/history) | View and manage query history |
| [`deep`](/en/reference/cli/deep) | Multi-agent deep research |
| [`sources`](/en/reference/cli/sources) | Manage search adapters |
| [`login`](/en/reference/cli/login) | Authenticate to a usearch server |
| [`repl`](/en/reference/cli/repl) | Interactive REPL mode |

## Global Flags

```
--help      Show help for any command
--version   Show version information
```

## Exit Codes

| Code | Meaning |
|---|---|
| 0 | Success |
| 1 | General error |
| 2 | Query pipeline error (no results, synthesis failed) |
| 3 | Authentication error |

## Output Format

By default, `usearch query` outputs Markdown to stdout. Progress and errors go to stderr.

```bash
usearch query "..." > result.md       # capture result
usearch query "..." 2>/dev/null       # suppress progress
usearch query --json "..." | jq .     # JSON output for scripting
```
