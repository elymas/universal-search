# Adapter Catalog

Universal Search uses a pluggable adapter system. Each adapter wraps a search source and implements the common `Adapter` interface (see [Architecture](/en/reference/architecture)).

<Callout type="info">
  This page documents the V1 adapter set. A plugin system for third-party adapters is planned in **SPEC-DOC-002**. See that SPEC for the adapter API contract and plugin development guide.
</Callout>

## Built-in Adapters

| Name | Category | Locale | Auth Required | Description |
|---|---|---|---|---|
| `searxng` | web | any | No | SearXNG metasearch engine (70+ providers) |
| `arxiv` | academic | any | No | arXiv preprints + Semantic Scholar |
| `github` | technical | any | No | GitHub repositories and issues |
| `hn` | social | any | No | Hacker News discussions |
| `reddit` | social | any | No | Reddit top posts |
| `youtube` | social | any | No | YouTube video transcripts |
| `naver` | korean | ko | Yes | Naver Search (Korean-locale) |
| `koreanews` | korean | ko | No | Korean news RSS aggregator |
| `social` | social | any | No | Cross-platform social aggregator |

## Adapter Details

### searxng

The primary web search adapter. Routes queries to SearXNG, which fans out to 70+ search engines including Google, Bing, DuckDuckGo, and specialized academic/technical engines.

- **Engine**: SearXNG (AGPL-3.0, runs as separate Docker container)
- **Rate limits**: Configured in SearXNG settings (see `deploy/searxng/`)
- **Configuration**: `SEARXNG_URL` environment variable

### arxiv

Academic paper retrieval from arXiv and Semantic Scholar.

- **Sources**: arXiv preprint server, Semantic Scholar API
- **Best for**: Computer science, physics, mathematics, biology papers

### github

GitHub repository and issue search.

- **Sources**: GitHub REST API v3 (public repositories)
- **Rate limits**: 60 req/hour unauthenticated; set `GITHUB_TOKEN` for 5000 req/hour

### hn (Hacker News)

Hacker News discussions via Algolia HN Search API.

- **Sources**: Hacker News (news.ycombinator.com)
- **Best for**: Tech community discussions, startup news

### naver

Korean-locale search via Naver Search API.

- **Requires**: `NAVER_CLIENT_ID` + `NAVER_CLIENT_SECRET` environment variables
- **Sources**: Naver News, Naver Blog, Naver Knowledge iN
- **Locale**: Korean (`ko`) only

See [Korean Locale Setup](/en/operators/korean-locale-setup) for configuration.

### koreanews

Korean news RSS aggregator (Naver-independent).

- **Sources**: Major Korean news outlets via RSS (조선일보, 한겨레, 경향신문, etc.)
- **Requires**: No API key
- **Locale**: Korean (`ko`) only
