# What is Universal Search?

Universal Search is a hybrid AI-powered search engine that combines traditional keyword search with modern vector embeddings for more intelligent results.

## Overview

Universal Search provides a unified search interface that:

- **Hybrid Search**: Combines exact keyword matching with semantic understanding
- **Multi-language Support**: Built-in support for English and Korean with extensible localization
- **Multiple Surfaces**: CLI, Web UI, Claude Code Skill, and MCP server integration
- **Self-hosted**: Complete control over your data and search infrastructure

## Key Features

### Intelligent Search

- **Semantic Understanding**: Finds relevant results even when exact keywords don't match
- **Contextual Ranking**: Results ranked by relevance to your query context
- **Multi-source**: Search across web, documents, and knowledge bases simultaneously

### Flexible Deployment

- **Docker Compose**: Quick local development setup
- **Kubernetes**: Production-ready Helm charts
- **Cloud-native**: Supports multiple deployment options

### Developer Friendly

- **RESTful API**: Easy integration with existing applications
- **CLI Tool**: Command-line interface for automation
- **Extensible**: Plugin system for custom data sources

## Architecture

Universal Search consists of three main components:

1. **Go orchestration plane**: Core search logic and API server
2. **Python sidecars**: Specialized services (LLM integration, indexing)
3. **Next.js web UI**: Modern web interface for search and management

## Use Cases

- **Enterprise Search**: Internal company knowledge base search
- **Research**: Academic and scientific literature search
- **Content Discovery**: Finding relevant content across multiple sources
- **Customer Support**: Intelligent FAQ and support documentation search
