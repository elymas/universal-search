# Getting Started

This guide will help you get Universal Search running in about 30 minutes.

## Prerequisites

Before you begin, ensure you have the following installed:

| Tool    | Minimum Version | Notes                           |
| ------- | --------------- | ------------------------------- |
| Docker  | 24+             | Required for dependencies       |
| Go      | 1.25+           | Required for binary build       |
| Python  | 3.11+           | Required for Python services    |
| Node.js | 22+             | Required for web frontend       |
| pnpm    | 9+              | Node package manager            |
| make    | Any             | Standard build tool            |

## Quick Installation

### 1. Clone and Setup

```bash
# Clone the repository
git clone https://github.com/elymas/universal-search.git
cd universal-search

# Copy environment template
cp .env.example .env

# Edit .env with your configuration
# Required: MEILI_MASTER_KEY, POSTGRES_PASSWORD, LLM_API_KEY
```

### 2. Start Dependencies

```bash
# Start all required services (Qdrant, Meilisearch, PostgreSQL, etc.)
make compose-up
```

This starts:
- **Qdrant**: Vector database for semantic search
- **Meilisearch**: Full-text search engine
- **PostgreSQL**: Metadata storage
- **Redis**: Caching layer
- **LiteLLM**: LLM provider integration

### 3. Build and Run

```bash
# Build the usearch binary
make build

# Verify installation
./cmd/usearch/usearch --version
# Expected output: usearch v0.1.0-dev
```

## First Query

Let's run your first search query:

```bash
# Basic text search
./cmd/usearch/usearch query "machine learning tutorials"

# Semantic search
./cmd/usearch/usearch query "how to learn ML effectively"
```

### Expected Output

```json
{
  "query": "how to learn ML effectively",
  "results": [
```

## Web Interface

Universal Search also includes a modern web interface:

```bash
# Start the web UI (in separate terminal)
cd web
pnpm dev
```

Visit `http://localhost:3000` to access the web interface.

## Common Issues

### Port Conflicts

If you see port conflicts during `make compose-up`:

```bash
# Check which ports are in use
netstat -tuln | grep LISTEN

# Modify ports in docker-compose.yml if needed
```

### Missing Environment Variables

```bash
# Ensure all required env vars are set
grep -v '^#' .env | grep -v '^$'
```

Required variables:
- `MEILI_MASTER_KEY`: Meilisearch master key
- `POSTGRES_PASSWORD`: PostgreSQL password
- `LLM_API_KEY`: Your LLM provider API key

### Dependencies Won't Start

```bash
# Check Docker logs
docker-compose logs

# Restart specific service
docker-compose restart qdrant
```
