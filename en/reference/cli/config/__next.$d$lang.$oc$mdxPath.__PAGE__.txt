# usearch config

> This page is auto-generated from `usearch config --help`.
> Run `make gen-docs` after CLI changes to keep it in sync.

```
Manage the usearch configuration file.

Configuration is loaded from an XDG-compliant TOML file with layered precedence:
  1. Command-line flags (highest)
  2. Environment variables (USEARCH_* prefix)
  3. Config file (~/.config/usearch/config.toml)
  4. Built-in defaults (lowest)

Usage:
  usearch config [command]

Examples:
  usearch config path
  usearch config show
  usearch config init
  usearch config get server.endpoint
  usearch config set server.endpoint http://localhost:9090

Available Commands:
  get         Print the value of a single config key
  init        Create a default config file
  path        Print the resolved config file path
  set         Set a config key value in the config file
  show        Print the effective merged configuration

Flags:
  -h, --help   help for config

Global Flags:
      --repl   force interactive REPL mode

Use "usearch config [command] --help" for more information about a command.
```
