View and manage available search adapters.

Usage:
  usearch sources [command]

Examples:
  usearch sources list
  usearch sources status
  usearch sources show reddit

Available Commands:
  list        List available sources
  show        Show detailed source information
  status      Show source health status

Flags:
  -h, --help   help for sources

Global Flags:
      --repl   force interactive REPL mode
