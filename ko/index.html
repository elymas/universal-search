Universal Search

Universal Search는 팀 규모의 소스 기반 연구 메타 에이전트입니다. 웹, 소셜, 학술, 기술, 한국어 로케일 검색을 단일 쿼리 플레인으로 통합하고, 결과를 하이브리드 LLM 합성을 통해 제공합니다.

빠른 탐색

  • Universal Search란? — 개요, 문제 정의, 사용자 페르소나
  • 시작하기 — 사전 요구사항, 설치, 첫 번째 쿼리
  • CLI 레퍼런스 — 7개 서브커맨드: query, config, history, deep, sources, login, repl (영어)
  • MCP 도구 — search, get_citation, list_sources, deep_research (영어)
  • 운영자 가이드 — 배포, 인증 설정, 관측가능성
  • 문제 해결 — 일반적인 문제와 해결책

주요 기능

기능설명
멀티 소스 검색웹, 소셜, 학술, 기술, 한국어 로케일
하이브리드 합성기본 요약+인용; /deep은 멀티 에이전트 심층 연구
CLI + MCP + 웹 UI다양한 워크플로를 위한 세 가지 인터페이스
한국어 지원한국어 로케일 검색 및 이중 언어 문서 (EN/KO)
자체 호스팅팀을 위한 Docker Compose 배포

5분 안에 시작하기

git clone <repo-url> && cd universal-search
cp .env.example .env        # MEILI_MASTER_KEY, POSTGRES_PASSWORD 등 입력
make compose-up             # Qdrant, Meilisearch, PG, SearXNG, LiteLLM 시작
make build                  # usearch 바이너리 빌드
./cmd/usearch/usearch query "universal search란 무엇인가?"

전체 설치 가이드는 설치를 참조하세요.
