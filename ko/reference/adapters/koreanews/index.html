Overview

koreanews 어댑터는 여러 소스에서 한국 뉴스를 집계합니다. 기본 설정은 주요 한국 언론사의 RSS 피드를 활성화합니다. 선택적 KoreaNewsCrawler(KNC) Python 사이드카로 추가 크롤링이 가능합니다.

핵심 특징:

  • locale.go: EUC-KR 레거시 피드를 UTF-8로 자동 변환
  • dedup.go: mecab-ko 형태소 분석 기반 근접 중복 감지 (Daum + KNC + 직접 RSS 3중 신디케이션 중복 제거)
  • knc.go: Python 사이드카 브리지 (USEARCH_KNC_ENDPOINT)

활용 사례: 한국 뉴스 모니터링, 한국 미디어 집계, 한국어 콘텐츠 트렌드 분석.

한국어 로케일 사전 요구사항:

  • EUC-KR 레거시 피드는 자동으로 UTF-8로 변환됩니다 (koreanews/locale.go)
  • mecab-ko 기반 중복 감지가 기본 활성화됩니다 — 여러 소스에서 발행된 기사가 하나로 통합될 수 있습니다
  • KNC 사이드카를 활성화하려면 USEARCH_KNC_ENDPOINT 필요
  • 인덱스 사이드 설정은 한국어 로케일 설정 참고

Setup

인증: 불필요 — 공개 RSS 엔드포인트.

어댑터는 기본적으로 공개 RSS 피드를 사용합니다. RSS 컴포넌트에는 API 자격증명이 필요하지 않습니다.

# 어댑터 연결 확인 (RSS 전용 모드):
usearch query --source koreanews "한국 AI 연구"

선택적 KNC 사이드카 설정:

변수기본값목적
USEARCH_KNC_ENDPOINT—KNC Python 사이드카 기본 URL (services/storm/koreanewscrawler/)
USEARCH_ADP009_ENABLE_KNCfalseKNC 사이드카 통합 활성화

한국어 로케일 사전 요구사항:

  • mecab-ko 형태소 분석은 dedup.go에서 중복 감지에 사용됩니다
  • 인덱스 사이드 한국어 토크나이징은 한국어 로케일 설정 참고
  • EUC-KR 레거시 RSS 피드는 자동 변환됩니다; 별도 설정 불필요
