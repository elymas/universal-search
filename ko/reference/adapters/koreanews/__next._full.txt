# Korean News 어댑터

<StatusBadge adapter="koreanews" />

Korean News 어댑터는 RSS 피드(기본 활성화), KoreaNewsCrawler(KNC) 사이드카(기본 비활성화), Daum(v0.1에서 robots.txt로 인해 비활성화)의 세 소스에서 한국어 뉴스를 집계하는 복합 어댑터입니다.

## Overview

`koreanews` 어댑터는 여러 소스에서 한국 뉴스를 집계합니다. 기본 설정은 주요 한국 언론사의 RSS 피드를 활성화합니다. 선택적 KoreaNewsCrawler(KNC) Python 사이드카로 추가 크롤링이 가능합니다.

**핵심 특징:**
- `locale.go`: EUC-KR 레거시 피드를 UTF-8로 자동 변환
- `dedup.go`: mecab-ko 형태소 분석 기반 근접 중복 감지 (Daum + KNC + 직접 RSS 3중 신디케이션 중복 제거)
- `knc.go`: Python 사이드카 브리지 (`USEARCH_KNC_ENDPOINT`)

**활용 사례:** 한국 뉴스 모니터링, 한국 미디어 집계, 한국어 콘텐츠 트렌드 분석.

<Callout type="info">
  **한국어 로케일 사전 요구사항:**
  - EUC-KR 레거시 피드는 자동으로 UTF-8로 변환됩니다 (`koreanews/locale.go`)
  - mecab-ko 기반 중복 감지가 기본 활성화됩니다 — 여러 소스에서 발행된 기사가 하나로 통합될 수 있습니다
  - KNC 사이드카를 활성화하려면 `USEARCH_KNC_ENDPOINT` 필요
  - 인덱스 사이드 설정은 [한국어 로케일 설정](/ko/operators/korean-locale-setup) 참고
</Callout>

## Setup

인증: 불필요 — 공개 RSS 엔드포인트.

어댑터는 기본적으로 공개 RSS 피드를 사용합니다. RSS 컴포넌트에는 API 자격증명이 필요하지 않습니다.

```bash
# 어댑터 연결 확인 (RSS 전용 모드):
usearch query --source koreanews "한국 AI 연구"
```

**선택적 KNC 사이드카 설정:**

| 변수 | 기본값 | 목적 |
|------|--------|------|
| `USEARCH_KNC_ENDPOINT` | — | KNC Python 사이드카 기본 URL (`services/storm/koreanewscrawler/`) |
| `USEARCH_ADP009_ENABLE_KNC` | `false` | KNC 사이드카 통합 활성화 |

**한국어 로케일 사전 요구사항:**
- mecab-ko 형태소 분석은 `dedup.go`에서 중복 감지에 사용됩니다
- 인덱스 사이드 한국어 토크나이징은 [한국어 로케일 설정](/ko/operators/korean-locale-setup) 참고
- EUC-KR 레거시 RSS 피드는 자동 변환됩니다; 별도 설정 불필요

| `Query.MaxResults` | 결과 수 제한 | 기본값 20 |

## Rate limits

- **광고 제한:** 0 (운영자 설정 피드 속도)
- **적용 메커니즘:** 운영자 설정 피드 크롤링 간격; 외부 할당량 없음
- **업스트림 할당량 문서:** 각 한국 언론사 RSS 피드 이용 약관 (언론사별 상이)
- **소진 동작:** RSS 피드에서 HTTP 429 반환 시 해당 소스에 `CategoryRateLimited` 반환, 다른 소스의 결과로 계속 진행

| RSS 503 | CategoryUnavailable | 언론사 일시적 다운 | 어댑터는 다른 소스 결과로 계속 진행; 한 언론사 다운이 전체 쿼리 실패로 이어지지 않음 |
