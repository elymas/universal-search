어댑터 카탈로그

Universal Search는 10개의 프로덕션 어댑터를 포함합니다. 각 어댑터는 검색 소스를 래핑하고 공통 Adapter 인터페이스(pkg/types.Adapter)를 구현합니다.

아래 카테고리 필터로 목록을 좁힐 수 있습니다. 공유 오류 카테고리는 오류 분류 체계 페이지를 참고하세요.

어댑터 라이프사이클

상태 배지는 어댑터의 라이프사이클 분류를 나타냅니다 (SPEC-DOC-002 D5):

배지의미
stable7일 롤링 성공률 ≥ 0.95
beta7일 롤링 성공률 0.80–0.94
disabled컴파일/플래그 게이트 스텁 — V1에서 실제 경로 없음 (현재: x)
deprecatedpost-V1 어댑터 제거를 위한 예약

한국어 로케일 어댑터

한국어 콘텐츠에 최적화된 두 어댑터:

  • Naver — NAVER_CLIENT_ID + NAVER_CLIENT_SECRET 필요; 한국어 UTF-8 쿼리를 그대로 전달
  • Korean News — 공개 RSS + 선택적 KNC 사이드카; mecab-ko 중복 감지

인덱스 사이드 토크나이저 설정은 한국어 로케일 설정을 참고하세요.
