# universal-search-web

Next.js 16 App Router frontend for Universal Search.

## Quickstart

```bash
# Install dependencies (from web/ directory)
pnpm install

# Start development server
pnpm dev

# Type check
pnpm typecheck

# Lint
pnpm lint

# Build for production
pnpm build
```

Note: `pnpm-lock.yaml` is generated by `pnpm install` and committed thereafter.
SPEC-DEP-001 will tighten lockfile enforcement in CI.
